## 🔧 HƯỚNG DẪN XÓA

### Cách 1: Xóa nhanh từng repo
1. Click vào link trong cột "Link Xóa" ở trên
2. Scroll xuống **Danger Zone**
3. Click nút **"Delete this repository"**
4. Nhập tên repository để xác nhận
5. Click **"I understand the consequences, delete this repository"**

### Cách 2: Xóa thủ công
1. Vào `https://github.com/handmadee/[repo-name]/settings`
2. Tìm mục **Danger Zone**
3. Click **Delete this repository**
4. Xác nhận bằng mật khẩu GitHub
